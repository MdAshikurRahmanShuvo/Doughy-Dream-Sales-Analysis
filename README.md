# Doughy Dream Sales Analysis (Excel Dashboard)

## 📊 Overview
This project presents an interactive sales analysis dashboard for a coffee & bakery business, "Doughy Dream". The dashboard provides clear insights into sales performance, profitability, and customer trends using Excel.

## 🎯 Objectives
- Analyze overall sales and revenue trends  
- Identify top-performing products and locations  
- Evaluate profit and cost efficiency  
- Enable dynamic filtering using slicers  

## 📌 Key Features
- 📈 Total Sales, Profit & Quantity KPIs  
- 🏪 Branch-wise performance analysis  
- 🍩 Product category insights  
- 📅 Date-wise sales trends  
- 🎛️ Interactive slicers for dynamic filtering  

## 🛠️ Tools & Technologies
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers

## 📂 Dataset Details
The dataset includes:
- Transaction ID  
- Date  
- Branch & Location  
- Product Category  
- Quantity Sold  
- Price, Cost, Revenue & Profit  

## 🚀 How to Use
1. Open the Excel file  
2. Use slicers to filter data (date, branch, category)  
3. Analyze KPIs and charts  

## 💡 Insights Generated
- Identified top-selling product categories  
- Compared branch-wise performance  
- Analyzed revenue vs cost trends  
- Highlighted high-profit segments
